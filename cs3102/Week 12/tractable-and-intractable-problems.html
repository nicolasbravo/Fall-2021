5 Tractable and Intractable Problems

  • Running time is essentially the number of steps needed for an input
    • Not a number, but a function that takes the input as an argument
  • \(TIME(n)\) is a complexity class that is the set of all functions where \(T(n) = n\)

5.1 Tractability

  • Tractable:
    • Feasible to solve in the “real world”
  • Intractable:
    • Infeasible to solve in the “real world”
  • Whether a problem is considered “tractable” or “intractable” depends on the use case
    • Tractable = polynomial time
    • Intractable = exponential time
